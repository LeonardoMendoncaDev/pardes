# PARDES

Metodologia hermenêutica de desenvolvimento orientado pela intenção, por Leonardo Mendonça.

> Peshat, Remez, Drash, Sod — as quatro camadas da intenção no software.

Site em `https://pardes.pro`.

## Estrutura

- `src/content/docs/` — conteúdo em Markdown/MDX
- `src/styles/custom.css` — estilo próprio mínimo
- `astro.config.mjs` — configuração Starlight

## Desenvolvimento local

```
npm install
npm run dev
```

Site local em `http://localhost:4321`.
